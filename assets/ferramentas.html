O QUE É MEU IP?

IP significa Internet Protocol e é um número que seu computador recebe quando se conecta à Internet. É através dele que seu computador é identificado globalmente.

Seu provedor define se seu IP será estático ou dinâmico. Conhecer seu IP é vital para segurança e suporte técnico.
